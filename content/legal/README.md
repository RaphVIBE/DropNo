# Documents légaux — Drop No.

Drafts français, structure défendable, à faire valider par un juriste e-commerce EU avant launch.

⚠️ **Aucun de ces documents n'a de valeur juridique en l'état.** Ils servent de base de travail pour gagner 5-10h sur la rédaction. Le juriste doit :
- Vérifier la conformité Code de droit économique BE
- Vérifier la conformité Directive 2011/83/UE
- Vérifier la conformité RGPD + APD (Autorité de Protection des Données belge)
- Vérifier la conformité 5e directive AML pour les transactions luxe
- Adapter selon la forme juridique (SA, SRL, SPRL) et la situation fiscale

Budget estimé pour validation complète : 1 500 à 3 500 €.
